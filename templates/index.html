About the App

This Flask app is designed to demonstrate a basic web page layout with Bootstrap integration. As a DevOps Engineer, you'll find that this app is the starting point for learning how to deploy, monitor, and maintain web applications in the cloud using CI/CD pipelines.

Features

  • Responsive design using Bootstrap
  • Flask back-end with Python
  • Simple deployment setup
  • API integration possibilities
  • Containerization with Docker
